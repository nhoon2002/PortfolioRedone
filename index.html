CodeFlo

Social network, fully-equipped in MERN stack.

...or head to the Github repo!

CodeFlo is a full-stack social networking application, built from scratch. This project was an ambitious attempt at a MERN stack social networking service, complete with rich features such as real-time chat, team builder, todo list, etc. My goal was to build a platform for all individuals who come across a great app idea, but do not have the man-power or expertise to make it come to fruition. CodeFlo hopes to make it easier for ideas to flow and keep flo'ing so that we'll keep cultivating cool new apps to shake up the world!

Check out the Live Site!

Sketch That

Multiplayer, real-time, real fun.


Using the power of Google Firebase, this project was an ambitious attempt at integrating web sockets, canvas, user authentication, and a functional chat system. One person gets to draw something, which the other player has to guess correctly in order to earn points to win the round.

MyBookLibrary

A social networking app connecting books to booklovers to each other.

...or head to the Github repo!

Using the power of jQuery, Google Firebase, and Google Books API, this project was my first web app featuring user authentication and database management. Once registered to the site, the user has the ability to populate his or her 'bookshelf' and show off the arsenal to others on our website!

See the Demo

MovieLibrary

Movie lovers, rejoice. Find out all you wish to know about the hottest movies, simply by searching for a keyword.

MovieLibrary is a social networking app for movie lovers. Based on MERN stack with Redux and Webpack, this app allows users to search for any movie in the database of all movies, see the cover posters, and (in future update) build a shelf of your favorite films.
